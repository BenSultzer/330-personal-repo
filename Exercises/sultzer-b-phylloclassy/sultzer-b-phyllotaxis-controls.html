About this app

  • The "Restart" button will reset the "flower" circle element generations to 0 and will start the app runtime timer over to refresh the trig values for the circle element radii variance. The button also clears the screen to black for a new drawing.
  • Changes made from starter code:
    • Rather than having a hard coded radius for each "flower" circle element, or replacing the radius with a constant variable, I decided to calculate the radius each frame according to a specific trig function. That way, the radius is always different and follows a wave pattern.
    • My control is a dropdown menu that allows the user to change which trig function is used to calculate the radius. The options are sine, cosine, or tangent.
